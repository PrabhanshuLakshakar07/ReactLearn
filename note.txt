jsx:-
What is JSX?
JSX stands for JavaScript XML. It allows us to write HTML-like code directly inside JavaScript. It’s the "Skeleton" of your React components.
🗝️ The 4 Golden Rules of JSX:
To build your Signup and Login pages without errors, remember these:

//1->The "One Parent" Rule (Fragments):
You cannot return multiple top-level elements. You must wrap them in a single parent, like a div or an empty tag <> called a Fragment.
2->JavaScript Inside { }:
Any valid JavaScript (variables, math, functions) must be wrapped in curly braces inside your HTML.
Attributes Change Names:
Since JSX is JavaScript, some HTML attributes are slightly different to avoid conflicts with JS keywords:

class becomes className

for becomes htmlFor

onclick becomes onClick (CamelCase is mandatory)

Self-Closing Tags:
In HTML, you could leave <img...> or <input...> open. In JSX, every tag must be closed.
